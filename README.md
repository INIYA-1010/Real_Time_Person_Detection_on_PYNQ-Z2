# 🚀 Real-Time Object Detection on PYNQ-Z2

> Design and implementation of a real-time embedded vision system on Xilinx PYNQ-Z2, demonstrating object detection, performance analysis, and hardware-software co-design principles.

---

## 📌 Project Overview

This project presents a real-time object detection system implemented on the Xilinx PYNQ-Z2 platform using Python, OpenCV, and embedded vision techniques. The system processes input images, detects human objects in real-time, and evaluates performance metrics such as latency, throughput, and execution time.

The project demonstrates the integration of software-based computer vision algorithms with an embedded FPGA-enabled platform, providing insights into edge computing and hardware-software co-design methodologies.
